 Proyecto: Modelado de Componentes Vehiculares en Java

Este proyecto corresponde a la actividad GA1-220501096-04-AA1-EV04 – Modelado de clases en Java: Componentes vehiculares y relaciones, en la cual se implementan diferentes clases que representan los principales componentes de un vehículo y sus relaciones usando Programación Orientada a Objetos (POO).

 Objetivo del proyecto: diseñar e implementar un modelo orientado a objetos que permita representar Un vehículo con sus datos principales, sus componentes individuales (motor, ruedas, chasis, luces, frenos)
Relación entre componentes:
Registro vehicular, Mantenimiento, Uso de sensores.
El proyecto permite practicar: Encapsulación, Modularidad, Composición entre clases, Buenas prácticas de Java.
